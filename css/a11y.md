# a11y





#### 대체 텍스트 **- 텍스트 아닌 콘텐츠는 그 의미나 용도를 인식할 수 있도록 대체 텍스트를 제공해야 한다.**

**명료성 - 콘텐츠는 색에 관계없이 인식될 수 있어야 한다.**

**- 지시 사항은 모양, 크기, 위치, 방향, 색, 소리 등에 관계없이 인식될 수 있어야 한다.**
